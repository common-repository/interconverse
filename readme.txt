== Description ==
<h4>InterConverse Campaign Link Shortener</h4>

Shorten long blog post URLs into InterConverse campaign links via <a title="InterConverse.com" href="http://interconverse.com">interconverse.com</a> that will be unobtrusively tagged on the bottom of each post.  Use the shortened url to easily share your posts on websites or social networking sites like Facebook, Twitter and Pinterest. No account or password is needed to use the InterConverse service through this Wordpress Plugin.

InterConverse provides a free url shortening service.  This Wordpress Plugin uses the InterConverse link shortening service.  On a Wordpress blog post action, the plugin captures the Permalink of the post and sends it to the InterConverse service.  Once the Permalink is shortened by the service, the service updates the original post with the shortened url by appending it to the bottom of the original post.

== Installation ==

1. Upload the directory 'interconverse' and its contents to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress.  That's it!

== Frequently Asked Questions ==

1. Where is the Shortened URL placed?

	*  On Post, the permalink url created for the post will be shortened to a InterConverse campaign link and appended to the bottom of your post automatically.

2. Can the InterConverse be deleted?

	*  Yes, you may edit the post and delete the url.

3. Can I create posts without the shortened URL being tagged to the bottom of each post?

	* Yes, just deactivate the plugin from the 'Plugins' menu in Wordpress.
